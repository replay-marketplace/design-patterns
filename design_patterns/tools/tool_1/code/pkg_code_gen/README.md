# pkg_code_gen

A Python package for AI-powered code generation.

## Installation

```bash
pip install -e .
```

## Function Signatures

- `code_gen(prompt: str, code_gen_dir_path: str, project_name: str) -> str` - Code generation
- `code_gen(prompt: str, code_gen_dir_path: str, project_name: str, template_dir_path: str) -> str` - Code generation with template
