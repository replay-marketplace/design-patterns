# Simple Python Functions

This project contains a simple Python file with example functions.

## Functions

1. `greet(name)` - Returns a greeting message for the given name.
2. `add_numbers(a, b)` - Returns the sum of two numbers.

## Usage

Run the script directly:
```bash
python simple_function.py
```

Or import the functions in another script:
```python
from simple_function import greet, add_numbers

print(greet("Alice"))
print(add_numbers(10, 20))
```
