# Simple Python Functions

This project contains a simple Python module with example functions.

## Functions

1. `greet(name)` - Returns a greeting message for the given name.
2. `add_numbers(a, b)` - Returns the sum of two numbers.

## Usage

```python
from simple_function import greet, add_numbers

print(greet("Alice"))  # Output: Hello, Alice!
print(add_numbers(10, 20))  # Output: 30
```

## Requirements

- Python 3.6+
