# API Signature

`merge_sorted_arrays(arr1: list, arr2: list) -> list` - Merges two sorted arrays into a single sorted array.

## Parameters
- `arr1`: First sorted array (list of comparable elements)
- `arr2`: Second sorted array (list of comparable elements)

## Returns
- A new sorted array containing all elements from both input arrays

## Example
```python
merge_sorted_arrays([1, 3, 5], [2, 4, 6])  # Returns [1, 2, 3, 4, 5, 6]
```
