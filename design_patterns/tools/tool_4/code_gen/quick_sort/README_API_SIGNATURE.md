# API Signature

`quick_sort(arr: list) -> list` - Sorts an array using the quick sort algorithm with partitioning and recursion. Returns a new sorted list.

`quick_sort_inplace(arr: list, low: int = None, high: int = None) -> None` - Sorts an array in-place using quick sort algorithm.

`partition(arr: list, low: int, high: int) -> int` - Partitions the array around a pivot element and returns the pivot index.
