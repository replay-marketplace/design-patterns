# API Signature

`bfs(graph: dict, start: any) -> list` - Performs breadth-first search traversal on a graph represented as an adjacency list, starting from the given node. Returns a list of nodes in BFS order.

`bfs_path(graph: dict, start: any, end: any) -> list` - Finds the shortest path between start and end nodes using BFS. Returns the path as a list of nodes, or empty list if no path exists.
