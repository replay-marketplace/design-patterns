# API Signature

## BinaryTree Class

`BinaryTree()` - Creates an empty binary tree

`insert(value: int) -> None` - Inserts a value into the binary tree

`search(value: int) -> bool` - Searches for a value in the tree, returns True if found

`inorder_traversal() -> list[int]` - Returns list of values in inorder (left, root, right)

`preorder_traversal() -> list[int]` - Returns list of values in preorder (root, left, right)

`postorder_traversal() -> list[int]` - Returns list of values in postorder (left, right, root)

## TreeNode Class

`TreeNode(value: int)` - Creates a tree node with the given value
