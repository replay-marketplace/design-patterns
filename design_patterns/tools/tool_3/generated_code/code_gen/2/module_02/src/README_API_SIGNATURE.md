# API Signature

## Function: mult

- **Parameters:**
  - `a` (int): First integer.
  - `b` (int): Second integer.
- **Returns:** int - The product of a and b.
- **Description:** Performs multiplication of two integers.

## Example Usage
```python
from module_02 import mult

result = mult(5, 3)
print(result)  # Output: 15
```
