# API Signature

## Function: add

- **Parameters:**
  - `a` (int): First integer.
  - `b` (int): Second integer.
- **Returns:** int - The sum of a and b.
- **Description:** Performs addition of two integers.

## Example Usage
```python
from module_01 import add

result = add(5, 3)
```
