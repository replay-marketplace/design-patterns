# Python function returning List[Tuple[str, int]]

Here are several ways to specify this type signature:

## 1. Using `typing` module (Python 3.5+)

```python
from typing import List, Tuple

def my_function() -> List[Tuple[str, int]]:
    return [("apple", 5), ("banana", 3), ("orange", 7)]
```

## 2. Using modern type hints (Python 3.9+)

```python
def my_function() -> list[tuple[str, int]]:
    return [("apple", 5), ("banana", 3), ("orange", 7)]
```

## Key Points:

- **Python 3.5-3.8**: Use `List[Tuple[str, int]]` from `typing`
- **Python 3.9+**: Use `list[tuple[str, int]]` (lowercase, no import needed)
- The tuple type is **fixed-length** with specific types for each position
