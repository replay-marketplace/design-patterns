# Input Prompt

Can you make, no, is it trivial and easy to make a webpage that creates tiles and within each tile in the grid it displays other webpages that you specify in a text box above them with an address bar?


---

# Anthropic Response

# Yes, it's quite trivial and easy!

You can create this with basic HTML, CSS, and JavaScript. Here's a simple example:

```html
<!DOCTYPE html>
<html>
<head>
    <style>
        body {
            font-family: Arial, sans-serif;
            padding: 20px;
        }
        #urlInput {
            width: 500px;
            padding: 10px;
            margin-bottom: 10px;
        }
        #addButton {
            padding: 10px 20px;
        }
        #grid {
            display: grid;
            grid-template-columns: repeat(auto-fit, minmax(400px, 1fr));
            gap: 20px;
            margin-top: 20px;
        }
        .tile {
            border: 2px solid #ccc;
            border-radius: 8px;
            overflow: hidden;
        }
        .address-bar {
            background: #f0f0f0;
            padding: 10px;
            border-bottom: 1px solid #ccc;
            display: flex;
            gap: 10px;
        }
        .address-bar input {
            flex: 1;
            padding: 5px;
        }
        iframe {
            width: 100%;
            height: 600px;
            border: none;
        }
    </style>
</head>
<body>
    <input type="text" id="urlInput" placeholder="Enter URL (e.g., https://example.com)">
    <button id="addButton">Add Tile</button>
    
    <div id="grid"></div>

    <script>
        document.getElementById('addButton').addEventListener('click', addTile);
        
        function addTile() {
            const url = document.getElementById('urlInput').value;
            if (!url) return;
            
            const tile = document.createElement('div');
            tile.className = 'tile';
            
            tile.innerHTML = `
                <div class="address-bar">
                    <input type="text" value="${url}" class="url-field">
                    <button onclick="this.parentElement.nextElementSibling.src = this.previousElementSibling.value">Go</button>
                    <button onclick="this.closest('.tile').remove()">✕</button>
                </div>
                <iframe src="${url}"></iframe>
            `;
            
            document.getElementById('grid').appendChild(tile);
            document.getElementById('urlInput').value = '';
        }
    </script>
</body>
</html>
```

**Features included:**
- ✅ Text input for URLs
- ✅ Grid of tiles that auto-arranges
- ✅ Each tile has an address bar
- ✅ Can navigate to different URLs within each tile
- ✅ Close button to remove tiles

**Note:** Many modern websites block being displayed in iframes for security reasons (X-Frame-Options headers), so sites like Google, Facebook, etc. won't work. But it will work fine for many other sites!
