# Cloud Services for Text-to-Video with Starting Image

Several cloud services offer prompt-to-video generation where you can specify a starting image:

## **1. Runway ML (RunwayML)**
- **Gen-2** model supports image-to-video
- Provide starting image + text prompt
- Cloud-based with API access
- Popular choice for professionals

## **2. Stability AI (Stable Video Diffusion)**
- Available through their API and DreamStudio
- Takes initial frame + conditioning
- Can be deployed on cloud platforms

## **3. Pika Labs**
- Offers image-to-video with prompts
- Accessible via web interface
- Supports starting frame guidance

## **4. Google Cloud (Vertex AI)**
- **Imagen Video** capabilities
- Support for conditional generation
- Enterprise-grade infrastructure

## **5. Replicate**
- Hosts various models including:
  - Stable Video Diffusion
  - Other image-to-video models
- API access with starting image support

## **6. Amazon Bedrock**
- Provides access to various foundation models
- Some models support image conditioning
- Enterprise cloud infrastructure

## **Most Accessible Options:**
- **Runway ML** - Most user-friendly, professional features
- **Pika Labs** - Good balance of quality and ease of use
- **Replicate** - Best for developers wanting API access
