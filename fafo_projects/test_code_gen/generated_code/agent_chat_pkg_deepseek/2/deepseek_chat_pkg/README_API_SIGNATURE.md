# API Signature

`simple_chat(prompt: str) -> str` - Send prompt to DeepSeek API, return response optimized for code generation

## Description

This function sends a prompt to the DeepSeek API configured for code generation tasks and returns the generated response.

## Parameters

- `prompt` (str): The input prompt/question to send to the DeepSeek API

## Returns

- `str`: The response text generated by the DeepSeek API

## Example

```python
from deepseek_chat import simple_chat

response = simple_chat("Write a Python function to reverse a string")
print(response)
```
