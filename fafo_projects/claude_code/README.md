# Simple Claude SDK Client Example

This is the simplest possible example of using the Claude SDK Client.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set your Anthropic API key:
```bash
export ANTHROPIC_API_KEY=your_api_key_here
```

## Usage

Run the example:
```bash
python simple_example.py
```

The script will send a simple "Hello, Claude!" message to Claude and print the response.
